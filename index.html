What is Grapel

Grapel is a system in which gamer's scores are registered. The purpose of Grapel is not only to register the score but people can also get something from this system such as money, ticket, gift card, etc.

Feature

Grapel is very simple but very powerful!

Grapel gathers the score from each game and creates the ranking table. But the most important function is the return. Grapel returns something (such as money, ticket card etc) to the player who ranks at the top. And Grapel is a cloud system, so almost every game can connect to this system if ever they need it.
