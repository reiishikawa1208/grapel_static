What is Grapel

Grapel is the system in which people can register the score. But not only registering the score but also people can get something from this system such as money, ticket, gift card and etc.

Feature

Grapel is very simple but very powerful.

Grapel gathers the score from each game and creates the ranking table. But the most important function is the return. Grapel return something (such as money, ticket card and etc) back to the player who ranks in the top. And Grapel is a cloud system, so almost the game can connect to this system if they need it.

how to use

Preparation is only creating an account.

At first, you create the account for Grapel in the following link, and then you only play the game which is against the campaign. When you play the game during the campaign, the score you played is registered into Grapel automatically. The campaign game is different each time.
